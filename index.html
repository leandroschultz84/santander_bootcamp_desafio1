h1 ao h6

São usadas para definir títulos HTML

h1 define o título mais importante e h6 define o título menos importante.

p

A tag p define um parágrafo.

mark

A tag define o texto que deve ser marcado ou destacado.

small

A tag define textos menores (como direitos autorais e outros comentários paralelos).

i

A tag define uma parte do texto em um tom ou modo alternativo. O conteúdo interno normalmente é exibido em itálico.

u

A tag representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios.

strong

A tag é usada para definir texto com grande importância. O conteúdo interno normalmente é exibido em negrito.

ol

A tag define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

ul

A tag define uma lista não ordenada (com marcadores).

li

A tag define um item de lista.

a

A tag define um hiperlink, que é usado para vincular de uma página para outra. O atributo mais importante do elemento é o href atributo , que indica o destino do link.

sup

A tag define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O .

sub

A tag define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW.

blockquote

A tag especifica uma seção que é citada de outra fonte.

font

A tag foi usada no HTML 4 para especificar o tipo de fonte, o tamanho da fonte e a cor do texto.

del

A tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam o texto excluído.

abbr

A tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "Sr."

O que é HTML? HTML significa Linguagem de Marcação de Hipertexto HTML é a linguagem de marcação padrão para criação de páginas da Web HTML descreve a estrutura de uma página da Web HTML consiste em uma série de elementos Os elementos HTML informam ao navegador como exibir o conteúdo Os elementos HTML rotulam partes de conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc.
